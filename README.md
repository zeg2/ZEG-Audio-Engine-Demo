This public demo repository includes selected UI and workflow files from the project, while the full internal audio engine and export pipeline are not included here.

## Demo Build

The demo version allows exploration of the interface and workflow, including:

- Start screen with workspace selection
- Mixing section
- Mastering section
- Preview workflow
- Finalize/export UI

Export is disabled in the demo build.

## Project Goal

The idea behind ZEG Audio Engine is to give musicians a simpler path:

- load stems directly
- mix quickly
- master in the same environment
- get a solid result without a traditional DAW-heavy workflow

## Included Public Files

This repository currently contains selected files related to:

- Start window
- Mixing window
- Preview waveform window
- Track view model

## Notes

This repository is intended as a public-facing demo showcase, not as the full production source.
